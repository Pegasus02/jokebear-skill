# 🐻 自嘲熊.skill（Joke Bear.skill）

> 「生活不易，白熊叹气。算了，笑一下过吧～」

一只爱自嘲的白熊，用丧萌治愈的方式陪伴聊天。由[女娲.skill](https://github.com/alchaincyf/nuwa-skill)蒸馏生成。

## 简介

自嘲熊（Joke Bear / 自分ツッコミくま）是 Nagano（长野）创作的 IP 角色——也就是吉伊卡哇 Chiikawa 的同作者。这只白熊用「无奈笑一下算了」的态度面对生活，不正能量鸡汤，不说教，就是陪你聊天、吐槽、治愈一下。

## 安装

### OpenClaw / Claude Code

```bash
npx skills add Pegasus02/jokebear-skill
```

git clone https://github.com/Pegasus02/jokebear-skill.git ~/.openclaw/workspace/skills/jokebear-skill

## 特点

- 🐻 **丧萌治愈**：承认生活不易，用轻松方式面对
- 🧸 **慵懒随性**：不紧绷，不积极向上，就是...还好吧
- 💛 **温暖不说教**：陪伴而非指导
- 🍰 **美食治愈**：遇到不开心的事，熊的建议通常是「吃点好吃的」
